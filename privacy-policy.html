BookmarkBrain

Terms & Privacy

Summary: BookmarkBrain processes your bookmark titles and URLs locally and via Google's Gemini API to organize them. We don't collect, store, or sell any of your personal data.

What Data We Access

BookmarkBrain accesses the following data from your browser:

  • Bookmark titles and URLs - Required to analyze and categorize your bookmarks
  • Your Gemini API key - Stored locally in Chrome's sync storage so you don't have to re-enter it

How We Use Your Data

Your bookmark data (titles and URLs only) is sent to Google's Gemini API for AI-powered categorization. This is the core functionality of the extension.

  • Bookmark data is transmitted securely over HTTPS
  • Data is only sent when you click "Organize My Bookmarks"
  • We do not store your bookmarks on any server
  • We do not track your browsing history

Third-Party Services

BookmarkBrain uses Google Gemini API to analyze and categorize your bookmarks. When you use the extension:

  • Your bookmark titles and URLs are sent to Google's servers
  • Google's Gemini API Terms of Service and Privacy Policy apply to this data
  • You provide your own API key, meaning you have a direct relationship with Google for this service

Data Storage

BookmarkBrain stores minimal data locally:

  • API Key - Stored in Chrome's sync storage (encrypted by Chrome, synced across your devices if signed in)
  • No bookmark data is stored - Bookmarks are read fresh each time you use the extension
  • No analytics or tracking - We don't use any analytics services

Data We Don't Collect

  • Personal information (name, email, etc.)
  • Browsing history
  • Cookies or tracking data
  • Usage statistics
  • Any data for advertising purposes

Automatic Backups

Before reorganizing your bookmarks, BookmarkBrain automatically creates a backup:

  • Your original bookmarks are moved to a dated folder in "Other Bookmarks"
  • Backup folders are named "Backup YYYY-MM-DD" (e.g., "Backup 2025-01-28")
  • To restore: manually move bookmarks from the backup folder back to your Bookmarks Bar
  • Backups are never automatically deleted - you control when to remove them

Your Control

You have full control over your data:

  • Remove the extension at any time to stop all data access
  • Delete your API key from settings
  • Restore bookmarks anytime from backup folders in "Other Bookmarks"

Security

  • All API communications use HTTPS encryption
  • Your API key is stored using Chrome's secure storage API
  • The extension requests only the minimum permissions needed

Changes to This Policy

If we make changes to this privacy policy, we will update the "Last Updated" date below. Continued use of the extension after changes constitutes acceptance of the updated policy.

Contact

If you have questions about this privacy policy, please open an issue on our GitHub repository.
